# Changelog

Tous les changements notables de ce projet seront documentés dans ce fichier.

Le format est basé sur [Keep a Changelog](https://keepachangelog.com/fr/1.0.0/),
et ce projet adhère à [Semantic Versioning](https://semver.org/lang/fr/).

## [0.2.0] - 2026-06-28

### Mis à jour
- README.md : passage au statut "En production (v1.0) / v2.0 en dev", ajout du site [mantota.com](https://mantota.com), de la stack Laravel 12 + Vue 3 + Tailwind v3 + Vite 7, et de la roadmap v2.0.
- `docs/architecture.md` : architecture v1.0 (Hostinger, MySQL, Redis, Cloudinary) et v2.0 (VPS/cloud, scalable), sécurité (score 8/10), scalabilité par niveaux.
- `docs/features.md` : modules v1.0 détaillés (Vendor, Influencer, E-commerce, Wallet, Feed, UGC, KYC, Admin, Ambassador, Parrainage).
- `docs/business-model.md` : modèle de revenus réel (5% campagnes, 20% retraits, 1.5% dépôts, 15% UGC), marché cible et projections.
- `docs/api.md` : endpoints internes v1.0 et roadmap API publique v2.0.
- `docs/faq.md` : FAQ enrichie avec les informations de production.
- `docs/ai-roadmap.md` : contexte v1.0 en production et planification IA v2.1.
- `ROADMAP.md` : phases v1.0 terminées, v1.x stabilisation, v2.0 scalable, v2.1 IA, v2.2 expansion.

## [0.1.0] - 2026-06-28

### Ajouté
- Structure initiale du dépôt public.
- README.md professionnel avec présentation, architecture, technologies et roadmap.
- Documentation complète dans le dossier `docs/`.
- Fichiers de gouvernance : LICENSE, .gitignore, CONTRIBUTING.md, ROADMAP.md, SECURITY.md, CODE_OF_CONDUCT.md.
- Dossiers `assets/`, `demo/` et `presentations/`.
